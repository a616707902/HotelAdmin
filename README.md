## 简介

HotelAdmin是基于Layui框架设计的Frame型酒店后台管理系统。其功能包括权限管理，角色管理，菜单管理，综合管理以及会员管理等。




## 组件说明

### Layui
目前基于Layui 2.2.6版本
官网： http://www.layui.com/

### echarts
目前基于echarts 4.0.2版本
官网： http://echarts.baidu.com/
